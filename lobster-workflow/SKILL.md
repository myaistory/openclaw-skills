# Lobster Workflow

Workflow management and automation system for Lobster coordination.

## Features

- Workflow definition and execution
- Step-by-step task automation
- Conditional branching and decision making
- Parallel task execution
- Progress tracking and reporting
- Error handling and compensation

## Usage

```bash
# Example usage
openclaw workflow --help
```

Configure workflows in your OpenClaw configuration file.
