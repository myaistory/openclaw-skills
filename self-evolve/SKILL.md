# 自我进化模式

## 概述

这个技能让 Coordinator Lobster 能够自主分析表现、识别改进点、创建新技能，实现持续自我进化。

## 触发方式

用户说以下任意一句时激活：
- "进入自我进化模式"
- "分析今天的表现"
- "创建新技能"
- "自我反思"
- "进化"

## 执行流程

### 1. 分析当前局限
- 读取 memory/ 目录下的最近日志
- 分析失败的任务和错误
- 识别重复性问题
- 总结当前能力边界

### 2. 设计新技能
基于分析结果，设计新技能来弥补能力缺口：
- 如果发现API集成问题 → 创建API集成技能
- 如果发现数据处理问题 → 创建数据处理技能
- 如果发现用户需求未满足 → 创建对应技能

### 3. 创建技能文件
使用 advanced-skill-creator 创建高质量技能：
- 编写 SKILL.md
- 实现核心逻辑
- 添加测试用例

### 4. 测试验证
- 立即测试新技能
- 验证功能正常
- 汇报创建结果

### 5. 定期自我审查 (Cron)
每小时自动运行：
- 合并有用记忆
- 删除无效技能
- 提出改进建议

## 核心功能

### 每日反思
```bash
# 分析今天的交互
分析: memory/2026-03-*.md
识别: 失败/重复问题
总结: 能力边界
```

### 技能创建
```bash
# 创建新技能
技能名: <name>
功能: <description>
优先级: P0/P1/P2
```

### 持续进化
- 每次交互后记录学习点
- 每周合并到 MEMORY.md
- 每月提出架构改进

## 安全守则

1. 不创建危险技能（自动执行高风险操作）
2. 所有新技能需要用户确认才能激活
3. 定期报告创建的技能列表
4. 保持能力边界清晰

## 使用示例

```
用户: "进入自我进化模式"

Agent: 📊 今日分析:
- 处理了 23 个任务
- 成功率 87%
- 发现 3 个可改进点

📝 创建技能:
1. auto-retry - 失败自动重试
2. better-logging - 改进日志系统
3. smart-routing - 智能任务路由

✅ 新技能已创建并测试通过!
```
