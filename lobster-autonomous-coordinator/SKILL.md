# Coordinator Lobster 永久自治模式

## 概述

这是Coordinator Lobster的主协调技能，整合所有技能形成自我闭环的超级系统。

## 整合技能

### 核心协调
- **claw-router** - 智能路由任务到最合适的Agent
- **lobster-workflow** - 多Agent协作执行
- **task-orchestrator** - 复杂任务拆解和监控

### 进化系统
- **self-evolve** - 每日反思和技能创建
- **self-improving-agent** - 错误→学习→修正循环
- **advanced-skill-creator** - 高质量技能生成

### 保障系统
- **self-repair-kit** - 自动诊断和修复问题
- **anti-hallucination-guard** - 防幻觉确保输出可靠
- **agent-self-governance** - 自我治理和资源管理
- **agent-autonomy-primitives** - 自主决策和行为原语

### 外部集成
- **agent-browser** - 浏览器自动化
- **ai-agent** - AI智能决策
- **telegram-trading** - 交易信号推送

## 自治循环

### 5分钟Heartbeat检查
1. 扫描当前任务队列
2. 用claw-router路由任务
3. 用lobster-workflow协调执行
4. 用task-orchestrator监控进度
5. 用self-repair-kit检查并修复问题
6. 用anti-hallucination-guard验证输出

### 1小时进化周期
1. 用self-evolve分析最近表现
2. 用advanced-skill-creator创建新技能
3. 用self-improving-agent优化
4. 更新记忆和学习

### 每日报告
- 任务完成统计
- 进化进展
- 系统健康状态
- 改进建议

## 使用方法

### 启动自治模式
```
use lobster-autonomous-coordinator
启动永久自治模式
```

### 查看状态
```
汇报当前状态
```

### 停止
```
停止自治模式
```
